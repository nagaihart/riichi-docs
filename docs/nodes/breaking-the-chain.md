# 連鎖の切断

人は、
ずっと忙しい状態が続くと、

「終わった」
「ここで止まっていい」

を感じにくくなることがある。

終わる前に次が来る。

片付ける前に次を始める。

休む前に別の仕事が入る。

それを長く繰り返すと、
身体は「止まるな」を覚えてしまう。


すると、

- 収穫前に次を植える
- 洗う前に次を作る
- 閉じる前に次へ向かう

ようになる。


問題は、
これが子どもへ連鎖することだ。

子どもは説明より先に、

- 焦り
- 空気
- 時間の流れ
- 止まれなさ

を見て育つ。


だから大切なのは、
「もっと頑張る」ではない。

一つ終わった景色を見せることだ。


- 収穫し終わった畑
- 空になったシンク
- 片付いた机
- 静かな部屋

それは、
“何もない”のではない。

閉じられた世界だ。


子どもは、
その静けさを覚えていく。


連鎖は、
怒ることでは切れない。

まず、
観測すること。

そして、
小さく閉じること。

人は、
完了した景色によって、
少しずつ「止まっても大丈夫」を学び直せる。
