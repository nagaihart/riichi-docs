    また、「369」のサウンド的な弥勒菩薩との一致にも話が進み、有名な写真も紹介された。
    私はその写真の指の形状からひらめいた。右手の折れた薬指と親指を1、立つ指を0とすると
    二進数(1001)が成り立つ。左手も同様に成り立つ、しかも十進数の９と１８の関係になる。
    以下このひらめきからテスラの曼荼羅の考察を試みる。

  + ２進数シフトとは
    1001 (2進数) = 9 (10進数)

    10010 (2進数) = 18 (10進数)         
    のように桁をシフト(２倍)すること、10進数なら１、１０、１００、、と同じ。

   + テスラの「369は宇宙の鍵」

    この2進数シフト＋ mod 9は、テスラの世界を浮かび上がらせてくる。
    具体的に現れてくることは以下となる。

    + {1,2,4,8,7,5}の循環

      この並び順が曼荼羅の細い線として出現する。
      
    + {5,7,8,4,2,1} 逆循環
      
      逆順も同じ
      
    + この６つの数字の形状は、アンフィニ(♾️)=無限の形状をしています
      
    + 表Bの、３回目、６回目、９回目の数字

      縦方向みると１から８が順序立てられてる、つまりリセットされ、外の外周円の細い線に戻る。
      
    + 3 ↔ 6 → 永遠に往復する二項リズム
      
      曼荼羅にはこの線の矢印はないが、矢印は9から６へ、９から３へ、６と３は双方向の矢印で書かれるべき

    + 9 -> 常に 0（完結・無の象徴）
      
    + 曼荼羅に結合線はないが、９から各数字をDG (Digital Root 後述)するとその数字になる

      ex.) 9 -> 5 => 14 =-> 5

           9が絡むとすべてその数字になる
           
  + おまけ、数字の系譜
    + スピではゲマトリア（Gematria）だけど、科学的な俺は以下のDGとする

    + DG (Digital Root)
      - 定義: 各位の和を繰り返し、一桁に還元する
      - 数学: mod 9 に基づく
      - 用途: 
        - 3, 6, 9 の宇宙リズムの抽出
        - 計算チェック、誤り検出
      - 宇宙的直感:
        - DG = 数の「基音」を抽出する
        - 9 = 死と生の統合点（不動点）
        - 3↔6 = 拍動の往復リズム

  + 一旦の結論

    エントロピーがほぼ無限の宇宙においてこんな初歩的な数術パズルの解法が通用するわけがない。
    茶番極まりない。


  + テスラと 3・6・9 の出所について

    + 事実
      - テスラの自著・特許・講演録には「3・6・9が宇宙の鍵」という表現は出てこない。
      - ただし彼は数字に強いこだわりを持っていた証言が多い。
      - 建物に入る前に3回まわる。
      - 18枚のナプキンを使う。
      - 3や9の倍数に執着する行動が目撃されている。

   + 出所
     - 「3・6・9は宇宙を理解する鍵」という言葉はテスラ本人の直筆資料にはない。
     - 後世の解釈、スピリチュアル文献、インターネットやYouTubeから広まったもの。
     - 元は彼の行動習慣に基づいた二次的な伝承。

   + まとめ
     - 3・6・9を宇宙の鍵とする思想は、テスラ本人の著作ではなく「後世の解釈」。
     - 数字に執着した事実はあるため、そこに現代的な数秘学や数論的アプローチを重ねて「369曼荼羅」が再構成されている。
